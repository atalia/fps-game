# FPS Game - 多人在线对战 FPS 游戏

多人在线第一人称射击游戏，基于 WebSocket 实时通信，Three.js 3D 渲染。

## 快速开始

### 1. 启动后端服务器

```bash
cd server
go mod tidy
go run main.go
```

服务器将在 `http://localhost:8080` 启动。

### 2. 访问游戏

打开浏览器访问 `http://localhost:8080`

或者使用独立 HTTP 服务器：
```bash
cd client
python -m http.server 3000
# 访问 http://localhost:3000
```

## 消息协议

### 客户端 → 服务器

| 类型 | 说明 |
|------|------|
| `join_room` | 加入房间 |
| `leave_room` | 离开房间 |
| `move` | 位置更新 |
| `shoot` | 射击事件 |
| `reload` | 换弹 |
| `chat` | 聊天消息 |
| `respawn` | 重生 |
| `weapon_change` | 切换武器 |
| `voice_start` | 开始语音 |
| `voice_stop` | 停止语音 |
| `voice_data` | 语音数据 |
| `team_join` | 加入队伍 |
| `grenade_throw` | 投掷物 |
| `c4_plant` | 放置 C4 |
| `c4_defuse` | 拆除 C4 |
| `skill_use` | 使用技能 |
| `emote` | 表情 |
| `ping` | 标记 |

### 服务器 → 客户端

| 类型 | 说明 |
|------|------|
| `welcome` | 欢迎消息（包含玩家 ID） |
| `room_joined` | 成功加入房间 |
| `player_joined` | 新玩家加入 |
| `player_left` | 玩家离开 |
| `player_moved` | 玩家移动 |
| `player_shot` | 射击事件 |
| `player_respawned` | 玩家重生 |
| `weapon_changed` | 武器切换 |
| `team_changed` | 队伍变更 |
| `grenade_thrown` | 投掷物 |
| `c4_planted` | C4 放置 |
| `c4_defused` | C4 拆除 |
| `skill_used` | 技能使用 |
| `chat` | 聊天消息 |
| `error` | 错误消息 |
